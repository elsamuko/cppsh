# cppsh

`cppsh` is a shell script that "interprets" single C++ files.

## Usage

    $ cppsh file.cpp

You can also initialize your `.cpp` file with the following line:

    #!/usr/bin/env cppsh

And then execute the file directly:

    $ chmod +x file.cpp
	$ ./file.cpp
